# Assignment 1 – The CSS Rockstar

This project was created for Assignment 1 in **26W Interface Design Using CSS**.

The goal of the assignment was to design a single-page website featuring the lyrics of a meaningful song using semantic HTML and custom CSS styling inspired by the song’s album cover.

## Song Information
- **Song:** God’s Plan
- **Artist:** Drake
- **Album:** Scorpion (2018)

## Project Features
- Semantic HTML structure (header, main, article, section, aside, footer)
- Custom CSS styling (no frameworks)
- CSS Grid layout
- Two Google Fonts (headings and body)
- Album-inspired black and white theme
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- GitHub Pages
